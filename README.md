# Einkaufsliste

Dieses Projekt zeigt meine ersten Erfahrungen mit Schleifen, Listen und Benutzereingaben in Python. 
Ich habe eine einfache Einkaufsliste erstellt, bei der man Artikel hinzufügen, entfernen oder anzeigen kann.

## Funktionen

- Artikel hinzufügen
- Artikel entfernen
- Einkaufsliste anzeigen
- Programm beenden

## Nutzung

Einfach das Script `einkaufsliste.py` in Python ausführen. 
Das Programm fragt nach Eingaben, um die Einkaufsliste zu verwalten.

## Beispiel

- Artikel hinzufügen: „Milch“
- Artikel hinzufügen: „Brot“
- Liste anzeigen → zeigt „Milch“, „Brot“
- Artikel entfernen: „Milch“
- Liste anzeigen → zeigt nur „Brot“
